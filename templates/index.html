Project Overview

Project Objective

Develop a comprehensive GPS-based vehicle tracking system that combines real-time location monitoring with machine learning-enhanced route prediction capabilities. The system provides accurate ETA predictions, route optimization, and comprehensive analytics for fleet management and logistics optimization.

Dataset Information

Source: Porto Taxi Trajectory Dataset from Kaggle

  • Records: 1.7 Million real-world GPS trajectories
  • Location: Porto, Portugal taxi operations
  • Data Points: GPS coordinates (latitude, longitude), timestamps, vehicle IDs, trip metadata
  • Coverage: Complete year of taxi operations with diverse routes and conditions

Machine Learning Models

Random Forest Regressor

  • Ensemble learning method for robust predictions
  • 100 decision trees with max depth of 20
  • Handles non-linear relationships effectively
  • Feature importance analysis for insights

XGBoost Regressor

  • Gradient boosting for superior accuracy
  • 100 estimators with learning rate 0.1
  • Optimized for speed and performance
  • Regularization to prevent overfitting

K-Means Clustering

  • 10 clusters for frequent pickup/dropoff zones
  • Identifies high-density traffic areas
  • Optimizes route planning strategies
  • Separate models for start and end locations

Core Algorithms

Haversine Formula

Calculates great-circle distances between GPS coordinates on Earth's surface with high accuracy

Linear Interpolation

Generates smooth GPS point transitions for realistic route visualization and animation

Bearing Calculation

Determines direction of travel between two points for route orientation analysis

Point-in-Polygon

Implements geofencing to detect when vehicles enter or exit defined zones

Feature Engineering

Geospatial Features

  • Start/End coordinates (latitude, longitude)
  • Total route distance (Haversine calculation)
  • Route bearing and direction
  • Route straightness ratio
  • Number of GPS points in trajectory

Temporal Features

  • Hour of day (0-23)
  • Day of week (Monday-Sunday)
  • Month of year
  • Weekend indicator
  • Rush hour indicator

Derived Features

  • Trip duration (seconds)
  • Average speed (km/h)
  • Start zone cluster ID
  • End zone cluster ID
  • Distance-to-duration ratio
